# 📈 Market Tracker Pro

A real-time, interactive stock market dashboard built with Python and Streamlit. This application allows users to track their personal portfolio, compare current prices against short-term and long-term moving averages, and visualize historical data.

* **Live Link:** (https://emwangi-market-tracker.streamlit.app/)

# ✨ Features
* **Live Market Data:** Fetches real-time stock and cryptocurrency pricing using the Yahoo Finance API.
* **Technical Indicators:** Automatically calculates and graphs the 7-Day and 30-Day Moving Averages to help spot market trends.
* **Custom Timeframes:** Interactive dropdown allows users to instantly shift the chart view from 1 month up to 5 years.
* **Smart Alerts:** Visual status indicators immediately flag if an asset is trading above or below its historical average.

# 🛠️ Built With
* **Python**  
* **Streamlit** (Frontend Web UI)
* **yfinance** (Data pipeline)
* **Pandas** (Data manipulation)

# 🚀 How to Run Locally
The app is connected directly to this GitHub repository. Thanks to Streamlit's continuous integration:
1. Any changes or bug fixes pushed to the `main` branch of this repository will automatically trigger a cloud rebuild.
2. The live website updates instantly within seconds of a `git push`, removing the need for manual server restarts.
